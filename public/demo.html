Host-Based Themes: The widget automatically detects the host and applies predefined color themes. Try changing the hostname in the URL to see different themes:
  • localhost - Development Blue theme (Blue + Green)
  • shop.example.com - E-commerce Red theme (Red + Teal)
  • Any other host - Default Gray theme (Gray + Amber)

Each theme includes complete color schemes: background, foreground, card, muted, accent, destructive, border, input, and ring colors.
